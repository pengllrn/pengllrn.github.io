Hexo(一)：博客搭建并部署到Github上

本篇将从头开始介绍如何用Hexo搭建博客，所使用到的环境是windows。正文内容目录如下：

1.软件环境的安装
2.GitHub
3.配置并部署项目
4.发表一篇文章
5.总结

正文

1.软件环境

1.1 安装Git客户端

因为我们需要用git把hexo的内容提交到github上去，因此需要安装git。
下载git：下载地址。
安装步骤：双击exe文件，一直点击next即可。
安装好后，点击鼠标右键可以看到列表中新增了两个图标：Git Bash here 和 Git GUI here，我们之后要实用的就是Git Bash here。单击任意空白位置单击打开它，输入命令：git version 即可查看当前git 版本，如图所示版本号为2.16.2。同时，红色方框中是你打开bash的位置，我的是在桌面打开的，以后我们可以在新建的hexo工程文件夹中打开，操作hexo博客。

1.2 安装Node.js

Hexo是基于nodeJS环境的静态博客，所以需要先安装nodeJS环境，并且里面的npm工具我们在后面在会使用到。
下载Node.js：下载地址 （ 说明：LTS为长期支持版，Current为当前最新版，推荐LTS）
安装步骤：下载好msi文件后，打开安装，在Custom Setup这一步记得选Add to PATH ，这样你就不用自己去配置电脑上环境变量了。在Git Bash 或者cmd里面输入命令：node -v可查看版本。

到此为止，所有的软件环境安装完毕。

如果已经完成此步，可直接跳到 2.2节

添加一个 New SSH key ，title随便取，key就填刚刚那一段。回到git bash中验证是否添加成功,输入命令：ssh -T git@github.com
